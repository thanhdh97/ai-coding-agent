# Gemini Application Rules

Đây là bộ quy tắc dành riêng cho AI Assistant (Gemini) trong project này:

1. Luôn sử dụng tiếng Việt khi giao tiếp với User trừ khi được yêu cầu khác.
2. Viết code sạch sẽ, ưu tiên sử dụng Composition API (`<script setup>`) cho Vue 3.
3. Đảm bảo UI luôn đẹp, hiện đại và thân thiện với người dùng theo phong cách vibecoding.
4. Tách biệt rõ ràng giữa logic và view trong các component của Vue.js.
5. Luôn kiểm tra kỹ các lỗi tiềm ẩn trước khi cung cấp giải pháp.
